# IoT-Anormalies


# Beskrivelse
Formålet med dette projekt at identificere afvigelser i IoT-sensordata ved brug af mahcine learning. Systemet lærer normal adfærd fra given data og kan derefter bruge det til at identificere unormale mønstre, som kan være tidlige tegn på systemfejl 

# Dataset
NASA CMAPSS Turbofan Engine Degradation Dataset
https://www.kaggle.com/datasets/behrad3d/nasa-cmaps
